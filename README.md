## Overview

A full-scale equity research financial model built on **Asian Paints Ltd (NSE: ASIANPAINT)** — one of India's largest paint manufacturers and a constituent of the Nifty 50.

The model covers **FY2016 to FY2025 historical data** with forward projections up to **FY2030**, built entirely in Microsoft Excel using publicly available annual reports and NSE/BSE filings.

---

## Model Components

| Sheet | Description |
|---|---|
| **Ratio Analysis** | 10-year profitability, efficiency, liquidity & cash flow ratios with sparkline trends |
| **Historical Financial Statements** | Cleaned P&L, Balance Sheet, Cash Flow Statement (FY16–FY25) |
| **Forecasting** | Revenue, EBITDA and EPS projections up to FY2030 |
| **WACC** | Peer comp beta derivation, cost of equity & debt build-up |
| **Intrinsic Growth** | ROIC, Reinvestment Rate and Intrinsic Growth Rate calculation |
| **DCF Valuation** | FCFF-based intrinsic value with sensitivity analysis |
| **Comparable Company Analysis** | EV/EBITDA and P/E peer benchmarking |
| **Football Field Analysis** | Valuation range across all methodologies |
